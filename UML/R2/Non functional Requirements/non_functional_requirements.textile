h1. Περιγραφή μη λειτουργικών απαιτήσεων

Όσον αφορά τις μη λειτουργικές απαιτήσεις, οι χρήστες προστατεύονται από κωδικό πρόσβασης τον οπόιο μπορούν να αλλάξουν όσες φορές θέλουν.
Το λογισμικό μας είναι έτσι διαμορφωμένο ώστε η επέκτασή του για έξτρα λειτουργικότητα μπορεί να γίνει εύκολα και γρήγορα.
Το σύστημά μας είναι βασισμένο σε java οπότε μπορεί να τρέξει σε οποιαδήποτε πλατφόρμα χρησιμοποιεί java.
Το σύστημα είναι έτσι διαμορφωμένο ώστε οι χρήστες να βρίσκουν γρήγορα τις λειτουργίες που χρειάζονται, χωρίς περιττά βήματα.
Μηνύματα λάθους εμφανίζονται αρκετά συχνά, οπότε η ελεγξιμότητα βρίσκεται σε καλό επίπεδο.
Επίσης για να συνδεθείς στην υπηρεσία πρέπει να έχεις λογαριασμό, οπότε δεν επιτρέπεται η σύνδεση σε μη εξουσιοδοτημένους χρήστες(ακεραιότητα).
Οι χρήστες δεν έχουν πρόσβαση στα προσωπικά δεδομένα των άλλων άμα δεν υπάρχει κάποιου είδους σύνδεσης μεταξύ τους.
